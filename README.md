## 🛠️ Installation Steps

1. Clone the repository

```bash
https://github.com/juanfdx/angular-adminPro.git
```

2. Change the working directory

```bash
cd angular-adminPro-main
```

3. Install dependencies

```bash
npm install
```

4. Run the app

```bash
ng serve -o
```

🌟 You are all set!
